EN: In the first step, run the xampp or wampp file or whatever you have and log in to phpmyadmin. Then download the Database.sql file and copy the contents of that file and paste it into the phpmyadmin sql file. Then configure your database in two files index.php and database.php. 

Important note: If you have xampp, everything is already ready for the initial settings of xampp and you can change it if needed.
